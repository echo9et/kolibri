# Kolibri Client

Легкое приложение TCP клиента на основе Qt, которое подключается к Kolibri серверу и отправляет сетевые данные.

## Обзор

Client — это минималистичный и событийно-ориентированный TCP клиент, построенный с использованием библиотек Qt Core и Network. 

## Аругменты командной строки (порядок неважен, может быть один из аргументов)
- `--host 192.168.0.100`  по умолчанию 127.0.0.1
- `--port 8080` по умолчанию 12345

## Зависимости

- **Qt 6.5**
  - `Qt::Core` - Основная функциональность
  - `Qt::Network` - TCP/IP сетевые возможности

Клиент будет:
1. Подключаться к серверу по адресу `ip:port`
2. Отправлять информацию об устройстве и метрики
3. Поддерживать соединение до прерывания
4. Попытки подключеняи каждый 5 секунд

## Архитектура

### Файлы

- **main.cpp** - Точка входа приложения, настройка подключения к серверу
- **client.h/cpp** - Реализация TCP клиента
  - Управление подключением
  - Передача данных
  - Обработка ошибок
- **randommessage.h/cpp** - Генерация тестовых данных для разработки

## Протокол данных

Клиент отправляет следующие типы данных (сущности описанны в ./Server/src/entities.h):
- Информацию о устройстве
- Сетевые метрики
- Лог событий


## CI/CD

Автоматическая сборка на GitHub Actions:
- Артефакты: `app-client` содержит исполняемый файл + библиотеки Qt
- Платформа: Windows x64 (win64_msvc2019_64)

## Сборка

```bash
# Из корневой папки проекта
cmake -S Client -B build/Client -DCMAKE_BUILD_TYPE=Release
cmake --build build/Client --config Release
```

## Использование

```bash
./Client
```
или
```bash
./Client --host 192.168.88.100 --port 8080
```
